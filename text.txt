First ground layer:
Состоит из тайлов, которые характеризуюттекущий биом. Доступные *нижние* тайлы состоят из coverType: 
0. свободный, планировал сделать oil ( нефть ); заперт передвижения по этому тайлу 0.0;
1. Вода; скорость передвижения 0.4;
2. Лава; запрет передвижения по этому тайлу 0.0;
3. Болото; скорость передвижения 0.5;
4. Земля; скорость передвижения 0.8;
5. Песок;  скорость передвижения 0.5;
6. Дорога; скорость передвижения 1;
7. Лес; скорость передвижения 0.7;
8. Камни( скалы, горы ); скорость передвижения 0.2;
9. Город; скорость передвижения 1;

Second ground layer:
Состоит из текстур, которые "вешаются" поверх нижнего уровня. Отличаются также по coverType. 
Функция находи совпадения и прорисовывает текстуры на местах.
